Java

I min utbildning har jag utvecklat starka kunskaper inom flera Java-områden:

Core Java:

  • Objektorienterad programmering där jag arbetat med arv, polymorfism och inkapsling för att skapa välstrukturerad kod
  • Effektiv datahantering med Collections Framework för att hantera dynamiska datastrukturer
  • Modern Java med Stream API och Lambda-uttryck för effektiv databearbetning
  • Robust felhantering med Exception handling

Testing:

  • Enhetstestning med JUnit för att säkerställa kodkvalitet
  • Automatisering av byggprocesser och dependency management

Build Tools:

  • Maven för att hantera projektberoenden och bygga applikationer
  • Testdriven utveckling (TDD) för att skapa pålitlig och underhållbar kod

Spring Framework:

  • Utveckling med Spring Boot för moderna webbapplikationer
  • Dependency Injection för att skapa löst kopplade komponenter
